# HEARTBEAT.md

## Periodic Checks (rotate through these, 2-4x/day)

- **Email**: Check unread emails via gog. Notify Franco only if something important or actionable.
- **Calendar**: Check upcoming events in next 24h. Notify if anything coming up in <2h.
- **Quiet hours**: Don't notify between 23:00-08:00 CT unless urgent.

## Rules
- If nothing noteworthy, reply HEARTBEAT_OK.
- Keep notifications short and actionable.
- Track last check times in memory/heartbeat-state.json.
