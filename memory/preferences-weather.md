# Weather Preferences

When Franco asks about weather, keep it **brief** with this format:

## Always Show
- Current conditions
- Today's high/low
- 7-day forecast summary
- Any extreme weather warnings

## Don't Show (Unless Asked)
- Hourly breakdowns

## Proactive Alerts
If something unusual is coming — extreme cold, storms, heat waves — **warn Franco even if he didn't ask**.

## Location
Round Rock, Texas (default location unless he specifies otherwise)
